// ----> Operator + If/Else <---- //

function cekDiskonBengkel(beliOli, totalBelanja, punyaMember) {
    let dapatDiskon = "";
    if ((beliOli === true && totalBelanja > 100000) || punyaMember === true) {
        dapatDiskon = "Selamat! Anda berhak mendapatkan diskon."
    }
    else {
        dapatDiskon = "Maaf, harga normal."
    }
    console.log(dapatDiskon)
}

cekDiskonBengkel(false, 20000, true);
cekDiskonBengkel(true, 150000, false);
cekDiskonBengkel(false, 1000000, false);

// ----> Penjelasan <---- //

Cara Berpikirnya (Translasi ke Mesin)
Bayangkan Bos Bengkel ngomong gini ke kamu:
"Yugata, tolong kasih diskon ya ke pelanggan yang beli oli dan belanjanya di atas 100 ribu. Oiya, ATAU kasih diskon juga ke orang yang punya kartu member."

Sekarang kita potong-potong instruksi Bos tadi jadi 3 bagian:

Bagian 1: "Beli oli DAN belanjanya di atas 100 ribu"

Kita terjemahkan: beliOli === true DAN totalBelanja > 100000.

Karena operator DAN itu &&, maka jadinya:
beliOli === true && totalBelanja > 100000

Trik: Karena ini adalah "Satu Paket Syarat", kita wajib bungkus pakai tanda kurung () supaya mesin memprosesnya sebagai satu kesatuan.
[Paket 1] = (beliOli === true && totalBelanja > 100000)

Bagian 2: "Punya kartu member"

Kita terjemahkan: punyaMember === true
[Paket 2] = punyaMember === true

Bagian 3: Kata penghubung "ATAU"

Bos bilang yang masuk kriteria Paket 1 ATAU Paket 2 berhak dapat diskon.

Operator ATAU adalah ||.

Hasil Akhir (Menggabungkan Bautnya)
Sekarang tinggal kita rakit:
[Paket 1] || [Paket 2]

Jadinya persis seperti kode yang bikin kamu pusing tadi:
((beliOli === true && totalBelanja > 100000) || punyaMember === true)
